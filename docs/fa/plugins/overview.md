# نمای کلی پلاگین‌ها

سیستم پلاگین امکان نصب بسته‌های ZIP قابل اعتماد را فراهم می‌کند. فایل‌های پلاگین بعد از نصب در مسیر زیر قرار می‌گیرند:

```text
var/plugins/{code}
```

رکورد `Plugin` اطلاعات code، type، status، manifest، مسیر، mainClass، permissions و خطای validation را نگه می‌دارد.

## وضعیت‌ها

- `installed`: پلاگین نصب شده ولی فعال نیست.
- `enabled`: پلاگین فعال و قابل استفاده است.
- `disabled`: پلاگین نصب شده ولی غیرفعال است.
- `error`: validation شکست خورده و پلاگین نباید استفاده شود.

نوع پلاگین پشتیبانی‌شده فعلی:

- `payment_gateway`

فقط پلاگین‌های قابل اعتماد را نصب کنید، چون کد PHP پلاگین داخل فرایند برنامه اجرا می‌شود.
