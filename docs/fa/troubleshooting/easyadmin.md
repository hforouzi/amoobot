# عیب‌یابی EasyAdmin

نکات و محدودیت‌های شناخته‌شده:

- `renderAsBadges` ممکن است در نسخه نصب‌شده EasyAdmin پشتیبانی نشود.
- `addPanel` برای بعضی layoutها قابل استفاده نیست.
- `CodeEditorField` ممکن است زبان JSON را پشتیبانی نکند.
- JSON array اگر مستقیم نمایش داده شود می‌تواند خطای array-to-string بدهد.
- `Action::NEW` را روی `PAGE_DETAIL` اضافه نکنید.
- مقدارهای i18n یا layout ممکن است null باشند؛ template باید این حالت را تحمل کند.
- فرم‌های RTL فارسی باید از نظر جهت، فاصله و چیدمان بررسی شوند.

در تغییرات ادمین، الگوهای موجود پروژه را دنبال کنید و هر دو زبان `fa` و `en` را بررسی کنید.
