# مستندات Amoobot

Amoobot یک فروشگاه VPN مبتنی بر تلگرام است که با Symfony ساخته شده و شامل بات فروش، پنل ادمین، اتصال به پنل‌های VPN، پرداخت، اتوماسیون و SDK پلاگین پرداخت است.
